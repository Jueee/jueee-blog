### 项目构建

在本地安装：

```shell
git clone https://github.com/Jueee/jueee-blog.git
cd jueee-blog
npm install
```

生成：

```shell
npx hexo generate
```

运行：

```shell
npx hexo server
```

清理：

```shell
npx hexo clean
```
