### 解决办法1

在 `C:\Users\Administrator` 中的`.gitconfig`文件，文本编辑器打开，记事本就可以，添加一行

```
[credential]
	helper = store
```

### 解决方法2

git bash进入你的项目目录，输入：

```
git config --global credential.helper store
```

然后会在本地生成一个文本，上边记录你的账号和密码。当然这些你可以不用关心。

然后你使用上述的命令配置好之后，再操作一次git pull，然后它会提示你输入账号密码，这一次之后就不需要再次输入密码了。
